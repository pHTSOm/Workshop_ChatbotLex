In this step, we will proceed to clone a website which we will use to connect with the chatbot.

1. Copy this ``https://github.com/pHTSOm/workshop_website_frontend.git``
2. Go to Visual Studio code
3. At the navigation bar choose **Source Control**
4. Click **Clone Repository**
5. Then paste the link we just copy.
6. Press Enter.
7. We can run it with npm install then npm start.
